Welcome ( ' 8 '*)

この言語は京都産業大学情報理工学部3年次秋学期開講のプログラミング言語の長期的な課題2で作成された言語です。 ( ' 8 '*) は母国語によらない言語を目指して作られました。そのため、言語の名前も顔文字を使っています。読み方は ' hiyo ' と呼んであげてください。生まれた小さなひよこにも満たない未熟さと可能性を表します。

<< この言語の特徴について >>

変数名に絵文字を使用することができます。例えば以下のように値を定義します。

🍀 ~ | 123 |

多くの言語では"変数名=値"で変数の値の定義が行われます。これは代入型である場合と束縛型である場合と統合型である場合があり、使用者に混乱をもたらします。また、現代は代入型から束縛型へと移行する時代でもあります。 そこで、束縛型であることを視覚的に意識してもらうために荷札をイメージした文法を設計しました。変数名には絵文字が使える以外に基本的にどんな言語でも使用できます。母国語が日本語の人でもドイツ語の人でもエスペラント語の人でも自由に変数名を指定できます。

<< この言語の目指す先について >>

多くのプログラミング言語は英語を軸とし、記述される事がほとんどです。英語を母語としない人にとっては英語を見ただけで辛い気持ちになる人も多いのではないでしょうか。英数字または記号の羅列が嫌悪感を与えることもあります。そこでこの言語では、printなどを絵文字または記号にし、ソフトな印象を与えます。これにより、絵文字の外観から何をする処理なのかを直感的に分かるようにしています。 多くの人がプログラミング言語に興味を持つきっかけになれることを目指して作成されました。 近年生み出されるビジュアルプログラミングほどビジュアルに依存せず、PythonやJava、Cといった言語に近い形で記述をします。ビジュアルに極度によらないことで、PythonやJava、Cといった言語への移行をしやすくします。この言語がプログラミングに興味を持つきっかけになることを目指して作成されています。
